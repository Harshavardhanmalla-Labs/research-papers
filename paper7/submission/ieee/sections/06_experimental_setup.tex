\section{Experimental Setup}

\subsection{Pre-Registration}

The cell grid, calibration strategies, hypotheses H1--H3, decision rules,
and stop rules in \texttt{paper7/design/PAPER7\_PROTOCOL.md} were locked
on 2026-06-04 before any online-calibration result was computed.

\subsection{Execution Order}

For each cell $K \in \{50, 100, 200\}$:
\begin{enumerate}
\item Cache per-window $(\text{pairs}_w, \text{HRS}_w)$ states for all
five calibration seeds (driving evolution with fixed-weight HygienePrio-full).
\item Cache per-window states for all twenty-five evaluation seeds
(same trajectory policy).
\item For each window $w = 1, \ldots, 6$, determine the
\textsf{fixed}, \textsf{online}, and \textsf{offline} weights as in
\S5.2, then score every evaluation seed at window $w$ with each
strategy's weights.
\end{enumerate}

\subsection{Statistical Reporting}

Cell-mean point estimates are means across the 25 evaluation seeds.
95\% confidence intervals are percentile bootstrap with 10{,}000
resamples~\cite{efron1994introduction}. No null-hypothesis significance
tests are reported; H1--H3 decision rules are applied as stated in the
protocol.

\subsection{Reproducibility}

The runner and analysis are released under MIT licence with the artifact.
From \texttt{paper7/}:
{\small\begin{verbatim}
PYTHONPATH=src python3 src/run_online_calib.py
PYTHONPATH=src python3 src/analyze.py
PYTHONPATH=src python3 src/make_figures.py
\end{verbatim}}
The runner re-uses Paper~5's \texttt{window\_sim} and Paper~4's
\texttt{hygieneprio} packages via \texttt{sys.path} injection (no
code duplication). Determinism is inherited from Paper~5's per-(seed,
window, step) RNG seeding.
