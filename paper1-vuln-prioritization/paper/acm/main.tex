\section{Reproducibility and Provenance}
All numeric results derive from a single frozen artifact (30 seeds, 13 strategies)
sealed with a content-addressed freeze manifest (per-file SHA-256) and verified
prior to table and figure generation. The run used bundled toy fixtures and the
deterministic synthetic fleet generator with no live feed calls and no production
data: 4{,}290 metric rows, 0 NaN, 0 infinite values, 390 hash-chain-valid audit
logs, maximum scheduled count equal to the configured capacity (100), strict
inspection passed with zero issues. See \texttt{../manuscript/reproducibility\_appendix.md}.
